Se trata de desarrollar una pequeña aplicación en el entorno de programación que te resulte más práctico y que debe cubrir los siguientes requisitos:

La aplicación es una pequeña calculadora.
Tendrá un campo para poner el nombre del usuario que realiza la operación.
Tendrá 3 campos para introducir los 3 operandos (Operand_1, Operand_2, Operand_3), que podrán contener valores numéricos o alfanuméricos.
Tendrá un botón para realizar la operación.

La operación que deberá realizar es:
Sumar el Operand_1 + Operand_2 + Operand_3, si los 3 operandos son numéricos.
Si cualquiera de los 3 operandos es alfanumérico, en lugar de sumar los 3 operandos, los concatenaremos.
Mostrará el resultado de la operación al usuario.
El Operand_1 y el Operand_2 son obligatorios; si el usuario no los introduce, el sistema deberá mostrar un mensaje indicando que tiene que informar un valor en ambos campos.
Cada operación que se realice se registrará en memoria, guardando qué usuario (introducido en el campo nombre de usuario) la ha realizado.
Si repetimos una operación que ya ha sido ejecutada por otro usuario anteriormente, además de devolver el resultado, devolveremos también el usuario que anteriormente había realizado esa operación.
